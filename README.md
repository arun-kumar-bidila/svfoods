# svfoods

A new Flutter project.

Backend Repo https://github.com/arun-kumar-bidila/svfoods_server

## Getting Started

SVFoods is a Flutter-based restaurant management application that enables real-time food ordering and restaurant workflow management. It provides both customer and admin/restaurant functionalities like browsing menu items, placing orders, tracking order status, and handling restaurant-side order management.


## Tech Stack

| Layer            | Technology                                                      |
| ---------------- | --------------------------------------------------------------- |
| Frontend         | Flutter, Dart                                                   |
| Backend          | Node.js, Express.js                                             |
| Database         | MongoDB                                                         |
| State Management | Riverpod                                                        |
| Auth             | JWT                                                             |

## Setup Instructions

Clone the repository:
git clone https://github.com/arun-kumar-bidila/svfoods.git
cd svfoods

Install dependencies:
flutter pub get


Run the app:
flutter run


Make sure backend server is running and API base URL is set correctly in your service/config files.
